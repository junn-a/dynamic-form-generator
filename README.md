# SmartPro – Dynamic Form Generator

Web application to generate and manage dynamic forms without coding.

## Features
- User login (PostgreSQL)
- Dynamic form builder
- JSON-based data storage
- Responsive UI (Bootstrap)

## Tech Stack
- PHP
- HTML, CSS, JavaScript
- Bootstrap
- PostgreSQL
- JSON Storage

## Folder Structure
assets/     -> CSS & JS
page/       -> Application pages
db/         -> JSON / database related files
uploads/    -> Uploaded files
